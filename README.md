# Simple Python Tkinter Calculator

A clean and modern GUI-based calculator built using Python's standard `tkinter` library. The design is inspired by modern, minimalist calculator interfaces, featuring responsive grid layouts and intuitive color-coded buttons.

## 🚀 Features
* **Elegant Design:** Customized color scheme with distinct button styles for operators, functions, and numbers.
* **Basic & Advanced Math:** Performs basic arithmetic calculations (+, -, ×, ÷), percentage logic (`%`), and square root functions (`√`).
* **Clean Layout:** Fully responsive layout that adapts to window resizing smoothly.

## 🛠️ Built With
* **Python 3**
* **Tkinter** (Python's built-in GUI toolkit)
* **Math Module**

## 💻 How to Run
1. Make sure you have Python installed on your system.
2. Clone this repository or download the `calculator.py` file.
3. Open your terminal/command prompt in the project directory.
4. Run the following command:
   ```bash
   python calculator.py
```
